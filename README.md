# Paradigma de Linguagens de Programação - Java

Este repositório contém os códigos, exemplos e exercícios desenvolvidos durante a disciplina **Paradigma de Linguagens de Programação**, utilizando a linguagem **Java**.

---

## 📘 Sobre a disciplina
A disciplina tem como objetivo estudar os diferentes paradigmas de programação, compreendendo seus conceitos, fundamentos e aplicabilidades.  
Durante o curso, a implementação prática foi realizada em **Java**, explorando os recursos da linguagem e a relação dela com os paradigmas estudados.

---

## 🛠 Tecnologias utilizadas
- **Java JDK 17+**
- **IDE IntelliJ IDEA** (ou outra de preferência: Eclipse, VS Code, NetBeans)
- **Git/GitHub** para versionamento de código
