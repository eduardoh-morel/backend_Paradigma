# Paradigma de Linguagens de Programação - Java + Spring

Este repositório contém os códigos, exemplos e exercícios desenvolvidos durante a disciplina **Paradigma de Linguagens de Programação**, utilizando a linguagem **Java** em conjunto com o **Spring Framework**.

---

## 📘 Sobre a disciplina
A disciplina tem como objetivo estudar os diferentes paradigmas de programação, compreendendo seus conceitos, fundamentos e aplicabilidades.  
Durante o curso, a implementação prática foi realizada em **Java**, com foco no uso do **Spring**, explorando a relação entre o paradigma Orientado a Objetos e frameworks modernos de desenvolvimento.

---

## 🛠 Tecnologias utilizadas
- **Java JDK 17+**
- **Spring Framework** (Spring Boot, Spring MVC, Spring Data, etc.)
- **Maven** ou **Gradle** para gerenciamento de dependências
- **IDE IntelliJ IDEA** (ou outra de preferência: Eclipse, VS Code, NetBeans)
- **Git/GitHub** para versionamento de código

---
